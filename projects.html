Overview:

Equal was an open project I did for my college Microprocessors course BU EC 450, in which my partner and I were tasked to come up with any project as long as it used the Texas Instruments MSP 430.

My partner and I are both music lovers so we decided to create a graphic equalizer. The device takes in analog stereo input and using a MSGEQ 7 chip it divides the audio spectrum into seven bands, detects their peak frequencies and multiplexes the output to provide a DC representation of the frequency at each band. The MSP 430 then takes in this dc representation and converts the values to a graphical representation on an lcd screen.

Overview:

Rock Paper Scissors (RPS) was an open project I did for my college Logic Design Course BU EC 311, in which my group and I were taksed to come up with any project we like on a Xilinx FPGA.

My team wanted to utilize as many peripherals as possible, so we decided to recreate the simple game of Rock Paper Scissors. We used the keyboard to input which choice the user wants to make, and a monitor to show the possible choices and who the winner was. All processing was done on the Xilinx FPGA where we used Verilog as our hardware description language.
